시간표 설정

지금 편집 중인 학기: 1학기

📅 학기·방학 기간 설정

매년 날짜가 조금씩 달라지니, 학교에서 안내받은 날짜로 맞춰주세요. 연도는 무시하고 월/일만 사용해요.

🏫 요일별 하교 시간

방학 기간에는 보통 등교하지 않으니 학교 시간은 생략했어요. 방학특강 등으로 등원 시간이 있다면 위 학기 탭에서 대신 학원 항목으로 추가해도 좋아요.

🎒 학원 시간표

같은 과목이라도 요일마다 시간이 다르면, 요일별로 항목을 따로 추가해주세요. (예: 수학 월 15:00~16:00, 수학 수 16:30~18:00 → 2개 항목)

🐙 GitHub 연동

토큰은 이 브라우저에만 저장되고, GitHub API 요청에만 사용돼요. 다른 곳으로 전송되지 않아요. Settings → Developer settings → Fine-grained tokens 에서 해당 저장소에 대해 Contents: Read and write 권한으로 발급하면 돼요.
